# 🚀 Full Stack Generative AI Projects

## 📌 Overview
This repository contains my hands-on implementation of concepts learned during a **Full Stack Generative AI Bootcamp**, covering end-to-end development of AI systems including **RAG pipelines, embeddings, transformers, and deployment**.

The focus of this repository is to build **real-world, production-ready GenAI applications**.

## 🔍 Key Projects

### 1️⃣ Semantic Search System
- Converts text into embeddings  
- Performs similarity search using:
  - Cosine Similarity  
  - Dot Product  
  - Euclidean Distance  
- Returns most relevant documents for a given query  

---

### 2️⃣ RAG-based Document Q&A System
- Document → Chunking → Embedding → Retrieval  
- Uses semantic similarity to fetch relevant context  
- Generates answers based on retrieved content  

---

### 3️⃣ NLP & Text Processing
- Tokenization using NLTK  
- Text cleaning and preprocessing  
- Feature extraction techniques
