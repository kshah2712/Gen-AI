## ⚙️ Setup

```bash
# Clone the repo
git clone https://github.com/kshah2712/Gen-AI.git
cd Gen-AI

# Create virtual environment
python -m venv venv
source venv/bin/activate        # Mac/Linux
venv\Scripts\activate           # Windows

# Install dependencies
pip install -r requirements.txt
```
